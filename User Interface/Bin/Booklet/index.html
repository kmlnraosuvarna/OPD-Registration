Forgot Password


1. If the User forgot the login Password, please click on "Forgot Password" option

2. Forgot Password screen will be displyed with required fields.

Enter UMR Number:


1. Enter the Patient SID in UMR field, which is given at the time of Patient Regisration

2. Then Click on "Continue" button

3. If the given SID Number is wrong it will display message "The entered UMR Number is wrong, please try again"


*After entering the SID Number, when the user will clicked on Continue button, it will display respective fields to reset the password

1. Get Verification code on my Phone

2. Get a Hint Question


Get Verification code on my phone:

*This option is used to get the verification code on patient mobile

1. When the user will select this option, it will display “Enter Registered mobile number” field with Continue button.



Get a Hint Question:

*This option is used to get the Hint question, which is selected at the time of patient changing the Password.

1. When the user will select this option,it should display Hint Question and Answer fields.

For the selection of first option:

Get a Verification code on mobile phone:


1. Select "Get a Verification code on my Phone" radio button

2. "Enter Registered mobile number" field will de displayed

3. Enter the Patient Regsiered mobile Number in "Enter Mobile Number" field

4. Click on "Continue" button


1. "We sent a verification code via text message (SMS) to your mobile" message will be dispalyed at the bottom

2. And along with this "Verification Code" field will be displayed with "Continue" button


1. If the given mobile number is correct, then only it will display "Verification Code" field with Continue button.

2. Enter "Verification Code" (Which is sent to your Registered Mobile Number) in Verification Code field

3. Click on "Continue" button

4. When the user will click on "Continue" button, New Password respective fields will be displayed

Wrong Mobile Number Entry:


1. If the given Mobile Number is wrong, it will display a pop up message with last 3 digits of patient registered mobile number

User Account Locking:


1. If the User will enter the Wrong Mobile number 3 times, then it will lock the process for 2hrs. with a pop up message saying that “Your account has been locked due to security reasons, please try after 2 hrs”.

2. After locking the process, if the user will try to do process before 2 hrs, it will display same pop up message with updated time.

For the selection of second option:

Get a Hint Question:


1. Select the "Get a Hint Question" radio button

2. Hint Question field will be displayed with the selected Hint Question, which is selected at the time of New Password creation.


1. Enter the the Answer in Answer field, which is given at the time of New Password creation.

2. Click on "Continue" button

3. When the user will click on Continue button it will verify the entered Answer is correct or not

4. If the given details are matched, it will display the New Password respective fields

Creating the New Password:


1. Enter New Password in “New Password” field

2. Enter same Password in “Confirm Password” field

3. Select required Hint Question from the “Hint Question” drop down

4. Enter hint Answer in “Answer” field (which is useful to get the password in future, if the user will forget the Password again)

5. Click on "Submit" button

6. "Password Changed Successfully" message will be displayed.

2. One pop up window will be opened with list of options

3. Select “Settings” option from the list


1. Settings window will be opened with list of settings options

2. Scroll down and Click on “Content Settings” button in “Privacy” Settings (Please see below attachment)


1. Content Settings window will be opened with list of options

2. Go to Pop-ups option and click on “Allow all sites to show pop-ups” option
